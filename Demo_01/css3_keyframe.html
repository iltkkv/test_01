1.首先在做圖片的時候，把每個影格的圖片做成一長條



2.

@keyframes titleAction {

　from { background-position: 0px bottom; } //原理是變換背景位置

　to { background-position:-7245px bottom; }

}

3.

#title:hover{

　animation: titleAction 1.5s steps(21) infinite;

　//step(影格數)；infinite：無限播放

}
